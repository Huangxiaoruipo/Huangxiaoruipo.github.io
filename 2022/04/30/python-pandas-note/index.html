pandas基本功能

使用的是tekla13.0版本套料工具生成工程时，形成的report数据，名字是myflie.csv

读取数据

pandas采用read_csv方法读取文件时会出现编码错误，在代码首部加上说明，读取时使用gbk的编码方式，全部代码如下：

# -- coding : utf-8 --
# coding: utf-8

import pandas as pd
import openpyxl
import numpy as np
import time
import xlrd

data = pd.read_csv('myfile.csv', header=None, sep=',', encoding='gbk')
print(data)

可以看到一共在myfile.csv内读取到了7行13列的数据，第一行是标题行，第三列到第十列被压缩起来了，我们无法看到数据的全貌，提供方法去展示数据：

在print之前添加一个pd.set_option('display.max_columns', None)的属性，顺便的把header设置为0，把第一行做为列索引（index_col=0可以设置列索引，usecols=[1,2,3]用来选定对应的列数）。

import pandas as pd
import openpyxl
import numpy as np
import time
import xlrd

data = pd.read_csv('myfile.csv', header=0, sep=',', encoding='gbk')
pd.set_option('display.max_columns', None)
print(data)

空值处理

在查看所有的数据以后，我们发现原始的数据的第二列（在Dataframe中是索引loc[0]），这时候我们对数据行进行一些处理，把NaN行先去掉，就可以在读取的时候直接跳过这行，Drawing No也是一个空数据，在后续的处理中去掉。两种种方法去处理NaN值：

读取的时候直接跳过

我们再用data.isnull().sum()去对数据表空值进行统计，方便后续处理

用dropna方法去除空行，指定 how='all'

data1 = data.dropna(axis=0, how='all', thresh=None, subset=None, inplace=False)#只有全部为空才回被删除
print(data1)

参数说明：

  • axis：默认为 0，表示逢空值剔除整行，如果设置参数 axis＝1 表示逢空值去掉整列。
  • how：默认为 ‘any’ 如果一行（或一列）里任何一个数据有出现 NA 就去掉整行，如果设置 how=’all’ 一行（或列）都是 NA 才去掉这整行。
  • thresh：设置需要多少非空值的数据才可以保留下来的。
  • subset：设置想要检查的列。如果是多个列，可以使用列名的 list 作为参数。
  • inplace：如果设置 True，将计算得到的值直接覆盖之前的值并返回 None，修改的是源数据。

假如这样行列处理完毕以后，我们数据还是有某些个值缺失，数据量大的时候可以这么做

data_isnull = data[data.isnull().values == True] #查找所有空值

很轻松就找到了整个表有缺失的值

另外补充查看各个列缺失情况的方法如下

data.isnull().mean() # 可以查看各列缺失值的比例

其他的:

基本运算
行间运算

在实例表中行间加法无意义，简单示例

加法

data[columns1]+ data[columns2]列之间相加columns是索引

对于行间所有数据求和，除了上述方法一个一个加外还可以

data.eval('合计=columns1+columns2+columns3')

或者

data.sum(axis=1)指定轴是列

将求和数据储存到dataframe中

data[合计] = data.sum(axis=1)

四则运算同理，减法，乘法，除法不再一一列出。

比较运算

data[columns1]><data[columns2]返回的是一组布尔值

也可以通过query()函数查看结果

data.query([columns1>[columns2])

同等于data[data[columns1]><data[columns2]]

data.query([columns1=='A')

同等于data[data[columns1]==[A]]

这两个方法都可以查看到比较运算的表结果

汇总运算

即对行之间的运算函数

count() 计数函数，会统计数据中非空个数

data.count()

类似len()和shape但是更加详细

省缺参数默认纵向统计，指定data.count(axis=1)可以横向统计列之间的空值

sum()

前面也展示了sum()的用法

默认省缺参数是对纵向求和，data.sum(axis=1)是对横向求和

mean()

平均数函数，类似sum()

可以指定内部参与运算的列和行，例如

data.mean(columns1，columns1)

除以上计算外，Pandas还支持最大值max()，最小值min()，另外进阶函数有：中位数函数median()、求众数函数mode()、方差函数var()、标准差函数std()、分位数函数quantile()和相关性函数corr()。

from PIL import Image
import numpy as np
a = np.array(Image.open(r"C:\Users\LENOVO\Desktop\21c55b005c746518e5e9f3522d9416b5.jpg").convert('L')).astype('float')
depth = 10
grad = np.gradient(a)   #获取图像灰度梯度
grad_x,grad_y = grad    #获取横纵梯度值
grad_x=grad_x*depth/100.    #横向归一整理
grad_y=grad_y*depth/100.    #纵向归一整理
A = np.sqrt(grad_x**2+grad_y**2+1.)
uni_x = grad_x/A
uni_y = grad_y/A
uni_z = 1./A    #得到归一单位法向量值

vec_el = np.pi/2.2  #建立光源三维坐标体系，对应横纵弧长
vec_az = np.pi/4.
dx = np.cos(vec_el)*np.cos(vec_az)  #获取光源具体位置，计算影响因子
dy = np.cos(vec_el)*np.sin(vec_az)
dz = np.sin(vec_el)

b = 225*(dx*uni_x+dy*uni_y+dz*uni_z)    #归一求出具体的光源影响量

b=b.clip(0,225)     #避免数据溢出

im = Image.fromarray(b.astype('uint8'))     #用framarry将得到的灰度转换为对应数组
im.save(r"C:\Users\LENOVO\Desktop\手绘.jpg")      #保存图形
